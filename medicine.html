Uses of Computer Science in Medicine

Introduction

Computer science has been used to advance many areas of modern medicine. For example:
  • Data Analysis
  • Medical Imaging
  • Computer Assisted Decision Making

Data Analysis

When dealing with data in medicine there tends to be large amounts, especially in medical research. This data needs to be processed and interpreted to analyse trends. There are many different processes that need to be applied to this data. For example:
  • Finding the mean, median and mode
  • Finding the standard deviation
  • Various tests of statistical significance
These processes, especially the statistical significance tests can be really time consuming. However, using a statistical package greatly reduces the time taken to perform these calculations. Statistical packages are made of four main components:
Biomedical Computer Package (BMD)
Provides a general set of advanced statistical operations
Statistical Package for the Social Science (SPSS)
Includes more statistical operations including multivariant analysis
Genstat
Specialises on variance analysis
Epi-info
Contains word processing and graphical functions

Medical Imaging

Computer science, can be used to generate 3D models of human anatomy, among other things. Specific hardware and software can be used to generate these images as well has high resolution CT scans and MRI's

Computer Assisted Decision Making

CMD is a interactive computer system that aids doctors and medical staff to diagnose patients and recommending treatment. It means doctors don't need to know everything in all medical fields, supporting their decision making. Link to home page
